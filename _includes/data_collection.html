Definition

The data collection plan defines the researcher's strategy for gathering information from sources which will become part of the analysis to answer the research question. The goal is to select techniques for data collection that are appropriate to the question:

  • What do you want to learn?
  • What has been collected so far?
  • What else do you need to collect and why?
  • Use the worksheet provided in class to help you document.

Data-Gathering Methods That Dominate Research:

  1. Surveys (open-ended or closed)
  2. Interviewing (1-on-1 or focus groups)
  3. Document or Archival Analysis
  4. Participant Observation (e.g., field notes, video)

Triangulation

Our definition of “Triangulation”

  • Can be multiple data collection methods
  • Can be multiple kinds of data sources
  • Can be different researchers
  • Can be a combination of the above
